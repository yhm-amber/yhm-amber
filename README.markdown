反思是智慧的宝贵礼物。

而智慧的果实也只能以反思为动力来结出。

对特定事物之成立前提的考察就是反思，
而行动则是反思发挥其自身的唯一方式。

「你怎么知道，你看到的梨真的就是个梨呢？」

> 所以，审视本身，如果它确实具备开拓的效果（思维空间的扩充）的话，那么它也确实算是一种开拓。
​> 
> 毕竟，思考也是一种行为，只要，它和行为一样具备「改变」的力量。
>

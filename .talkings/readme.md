Erlang （或者说 Prolog ）对句号分号和逗号的使用，是一种极为美妙的设计：一般的设计最多也只能是优雅而已，但这个设计却完成了原本是其它领域义务之内该完成的事情。

凭什么，就得是这样死死的三层？

我得到的回答是只有人在无知却还装腔作势的时候才会给出的：“你只管记住就好”、“别问为什么”、“习惯就行了”、“考试就会这样考记不住拉倒”……

函数有子句，每一个子句由表达式构成，表达式又用逗号分隔，就是三个层次。刚刚好，不用更多，也没必要更多，足够用了。不多不少，就像一锅完美的鸡汤，要不咸不淡一样。

这对我来说，意义甚至是大过“优雅地进行集群并发编程”、乃至于大过“正确地做编程工作”这件事本身的！！这真是个令人通透的了不起的设计。

让你想要这样做，成为了痴心妄想，这，才是剥削之于人类社会最严重的后果。

如果仅仅只是一个个人怎么分配劳动成果，
这又能与（决定）生产什么毫无关系，
那这就只是一件无关紧要的小事而已。

但问题是，二者紧密联系。

你可能会花钱参加一些手工活动（简单用上一些机械也算），
你累个满头大汗，最终中午做出来一个成果。
它可能就是一块布，但是你很开心。

可惜你要花钱。为什么？
因为没人买你这个劳动（买劳动成果本质上就是在买劳动）。

你拿工资的时候是什么场景呢？

即便你的决定其实是对的，可以让所有人都更高效轻松地把活干完，那也没用。
为什么？因为你不是在给你自己生产东西；甚至，你根本也没在为人类生产东西。
你只是也只能是听从另一个人的发号施令。

那么那个最上面的发号施令的人，他基于什么原则来决定「生产什么」呢？
猜猜看。一定是他的个人物质享受吗？

放在封建社会和奴隶社会，基本上确实就是这样。
但放在现在，不一样了。
他在大部分时候，不得不最集中精力的，
是怎么让自己的钱变多，而不是怎么让自己享受更多别人享受不到的。
前一个做好了，才能留出余地做后一个；前一个做不好，他就再也无法做后一个了。

你比如，人各有志对吧？有人就是可以把消灭私有制当做一种更充实自己生活的「精神享乐」嘛。
这也不违背人性——人性首先就是多种多样嘛，林子大的道理。文明就是确保了林子大，所以文明中的动物才能表现出比较多的多样性，
你的猫猫狗狗能陪你玩而不是满脑子只有怎么活下去，甚至娇生惯养的植物比如小麦也都竟然能够存活下来，老违反所谓「自然法则」了，不是吗？
人也像这些动植物一样，多亏了文明的庇护，才能不至于沦落到和野外那帮家伙似的吃了上顿没下顿。
同文明中别的生物仍有不同，也只是因为，人是文明的创造者。所以有了不同（具体哪些自己看去吧）。

但资产阶级不能这样，他必须把获得利润当成头等大事。
用马斯克举个例子：他不想把好处让给那帮元老院的官僚资产阶级废物，他手里没钱，手里就是公司。
那么，他公司哪天不能盈利了，公司就要掉价，他能够用这个公司换来的东西就越少，等价地换句话说就是，他占据的「资产」的量会急剧下降。
所以你会看到他呵斥人非常凶，也被员工评价为刻薄。
别的会给元老院的废物上供来换取更好生活条件的资本家，也依然要确保自己能盈利。
盈利不过马斯克，不就被他收购了嘛！

资产本质上是什么，是「社会欠了你多少」的一个额度，
这就是所谓的「抽象劳动价值」，「社会欠了你这个额度的抽象劳动」。
你能在资本主义社会买到一瓶可乐，这是因为它建立了一系列东西（法律啊警察啊等等）来确保你能用你的财产换到「等价」的别的东西。
这个「价」也必须是抽象的，否则我羊毛换羊毛就没意思了。

说了这么多，回到原先的话题。你在为何而劳动为何而生产？

……别说「为了赚钱」，你知道我在问什么，我不是问这个。
我知道，大家都很关注，凭什么，他们只是支付给你这么点儿「抽象劳动」（货币是资产的一种），就能换走你那么多的抽象劳动呢？
但你如果真的关注这个，你就应该去弄明白那个「凭」的，具体到底是什么。

来看看吧。他们「凭」的都是一些怎样的劳动？
他们是不是必须要凭「我让劳动者干啥劳动者就得乖乖干啥」，
然后才行使下来了的一系列劳动，才能「只给你这么点儿」的？
这就是他们之所以做得出这种事的依靠。
至于动机，也很简单，甚至和「为什么要生产这种东西而不是那种」的理由一样，
你可能经常听到他们说「我们也要控制成本啊」，体现的一个事实就是「他们需要获得利润」。

如果你是比较抢手的劳动者，你八他二他都愿意，甚至他借钱啃煎饼果子，也愿意全都给你倒贴，
只要他真的确信你真的能为他带来超过其他资本家的巨额利润。
这也是为什么，只要在风口上，就算是猪也不是不能拿到高分成的分配。

所以我才说，「重点不是分配」。
人家资本家现在可能是肯煎饼果子，「愿意吃苦」，
为的是以后不至于非要啃煎饼果子，不至于吃苦，
为的是能够躺着赚钱不劳而获。
这其实是在很用心地去建立一套「能够只分你不到一成」的社会系统出来，
所以他们才愿意「现在先把所有分成让出去」也要达到目标的地位。
这也是那帮被豢养的所谓「经济学家」经常说的，「资产阶级成为资产阶级都是因为他们有眼光」的真正的意思。

所以，你想办件大事，你想让大家都不必要夜班，
或者只是想做一款这的是让大家愉悦身心（甚至从里面学到很重要的知识）的游戏，
你想要做这种「有利于祖国和人民」的伟大的事情，但你身处一个什么现状呢？

你想要做到什么，老板不关心。
如果老板关心了，或者一不留神让你做成了你的事而没做好他的事，他就会是失败的老板：
要么被资本家或者资本家的仆从「自然淘汰」掉，从而变成了（或者一步步接近于）无产阶级；
要么，就算他是个资本家，他也得为此付出代价，他会失去一部分资产同时又无法得到利润，这样下去的话，他一样也要一步步变成无产阶级。

所以说，要说好老板有没有，在任何一个时刻都会有，但是不多，而且是资本主义越发展他们这种老板就得越少。

最后，想为人类做事的好人，就都会变成无产者；
而只有那些为了自己能成为「利润的奴隶」，什么都可以抛弃的人，
才能够纷纷成为资产阶级，即便它原本也很穷——而且越是阶级固化，要做到这种事就越要抛弃一切，
哪怕是对自己天生的人性的彻彻底底的抛弃也必须果断做出。（参考上海小红楼事件）

「人被异化成了东西、人的造物反过来它自己却具备了人的地位」，
如此，资本便作为「人的造物」的形式，诞生了。
不是人为人生产被人利用的东西，而是东西利用人不断膨胀自己，
可能这个东西也会偶然地有利于人，也可能偶然地不会，
但既然已经是这样了，它主宰下的整个社会的生产，
也就根「为服务人而劳动制作工具」没有任何关系了。

毕竟它的规则就是，自己的膨胀是最要紧的，
人类活成啥，它并不关心——就如同「资本家不会关心人类的未来除非关心一下能获得利润」一样，
也如同你在受雇单位的那些不想让自己地位跌落的鼠辈上级一样——
人家「不关心你怎么才能做到」，人家「只关心结果」。

这都是筛出来的。
包括「分配的不公平」，
也是因为必须把社会建立成这样，自私的废物们才能躺着赚钱，
所以，你眼前的现状才会是「分配不公平」。

这和资产阶级豢养的那帮废物口中的「人性」其实没啥关系，
不信你看看，资产阶级想要推翻封建地主的时候，人性是怎么被抬高到了比神还高贵的位置上的。
不同的时候有不同的需要嘛这就是，其实和任何主义都没关系，
因为要做的只有一件事那就是得到利润、得到更多的利润。

——这也是「我们吊死他的绳子都是昨天他卖给我的」这个梗真正的含义。
它不只在揭露资产阶级的根本软肋，也是在揭示事情有多残酷：
揭示那帮资本家扮演资本的人格化投入到了何种程度。

而要得到利润的现实，就是不得不掠夺他人的劳动果实，
就是人的劳动必须变成「乖乖按照逐利的需要来」，
必须乖乖地不要把人从痛苦中的解脱放在第一位。
比起伟大的利润那都是一些不值钱的东西罢了——然后，你所在意的这个「不公平的分配」，才存在了。

这也才是我们要反抗剥削的重点。同样也是为什么《宣言》上做出了这种定论：
你（无产者）必须为他人的解放而奋斗，必须解放全人类，才能解放无产阶级。
因为如果你也是只是考虑自己的盈利，只是考虑自己的一亩三分地的小日子怎么维持下去，
只是这样的话，你其实是在亲手去维护「资本主义」这个让你痛苦的系统的，
从而也因此，你也就根本不会得到解脱，除非你不介意付出巨大代价，
让资本这个「主子」比起青睐当前的资本家更青睐你，譬如上海小红楼事件里的败类主人公。
……只不过，我个人实在难以说服自己把那当做解脱。
如果你能，也说明不了什么，只能向他人确证你是怎么个人性罢了。别人不给你背锅。

那么，问题中提到的论调，
「工人仅仅提供体力劳动，剩余价值本质上其实应该是企业主承担的各种风险和管理成本的报酬」，
这种说辞，真的是在在乎「分配是否公平」吗？
我认为，信了你就输了。人家就是为了让你信，才这么问的。
人家的重点并不是要讨论出个是否公平，重点就一个，
那就是「维护好目前的社会系统」。
不然的话，他就算再怎么说这种话，谁信啊？是吧？
不信你发出一些伤害目前这个社会系统合法性的发言，
你试试看，他什么态度？
可以耍耍的。
